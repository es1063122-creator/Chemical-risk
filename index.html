화학물질 위험성평가 스튜디오 별도 사이트형 · 한국어/일본어 · SDS 자동분석 · PDF 보고서

별도 사이트 구조로 바로 확장 가능한 화학물질 위험성평가 웹

실제 화학물질 DB 검색, CAS 자동완성, SDS PDF 읽기, 위험도 계산, 보고서 반영, PDF 출력까지 한 화면에서 동작하는 운영형 버전입니다.

1. 화학물질 자동 DB이름 또는 CAS 번호로 검색하고 자동으로 물질 정보를 채웁니다.
2. SDS 자동 분석 엔진GHS 분류, 노출경로, PPE, 저장조건, 권장조치를 자동으로 제안합니다.
3. 위험도 AI 계산유해성·위험성·취급량·환기·작업시간·빈도를 합산하여 위험레벨을 계산합니다.
4. PDF 보고서 자동 생성입력된 내용을 그대로 PDF로 저장할 수 있습니다.
5. 한국어 / 일본어한국어와 일본어만 전환되도록 구성했습니다.
6. 실제 DB 연결형window.CHEMICAL_DB 외부 파일이 있으면 자동으로 연결하고, 없으면 데모 DB로 동작합니다.
