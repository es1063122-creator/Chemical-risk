화학물질 위험성평가 스튜디오 별도 사이트형 · 다국어 · SDS 자동분석 · PDF 보고서

별도 사이트 구조로 바로 확장 가능한 화학물질 위험성평가 웹

이 버전은 대표님이 요청한 핵심 기능을 한 화면에 넣은 베이스입니다. 화학물질 DB 검색, SDS 위험성 자동 분석, 위험도 점수 계산, 다국어 전환, PDF 보고서 출력까지 바로 테스트할 수 있습니다.

1. 화학물질 자동 DB이름 또는 CAS 번호로 검색하고 자동으로 물질 정보를 채웁니다.
2. SDS 자동 분석 엔진GHS 분류, 노출경로, PPE, 저장조건, 권장조치를 자동으로 제안합니다.
3. 위험도 AI 계산유해성·위험성·취급량·환기·빈도를 합산하여 위험레벨을 계산합니다.
4. PDF 보고서 자동 생성입력된 내용을 그대로 PDF로 저장할 수 있습니다.
5. 다국어 지원한국어, 일본어, 영어, 베트남어 전환이 가능합니다.
6. 별도 사이트 배포형chemical-risk 전용 사이트로 GitHub Pages 또는 Firebase Hosting에 배포하기 좋습니다.
